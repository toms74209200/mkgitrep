# mkgitrep

Make git repository.

## How to Use

**Please replace `fullname` and `address` of this script.**

Work script.

```bash
/foo$ sh ./mkgitrep.sh
select license:
```

Select license.
- `mit`(MIT License)
- `Apache`(Apache License)
- `none`(The Unlicense)

Make LICENSE.
Top of this file is described repository name picked from **current directory**.
```bash
/foo$ cat ./LICENSE
MIT License

foo Copyright (c) 2019 fullname <address>
...
```

Make README.md.
This file is described contents as follows.
- Repository
- License
- Author
